Excel Project

Quick data cleaning and visual representation project using Microsofe Excel.

Data Cleaning

The first issue that had to be taken was to clean the data so that it could easily be read as well as visualized. If I want the data to be easy to differenitiate i immediatly see that the marriage status must be changed as well as gender to make the data look more appealing and straight forward I do this using find and replace. As well I used IF statements to put the age into ranges as appose to specific points

Pivot Tables And Dashboard

Once the data is clean it is easy to put into pivot tables and from there the visualization is Completed.
